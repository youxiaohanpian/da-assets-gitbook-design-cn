# Import Issues（导入问题）

## 常见问题

本文档介绍在使用 Figma Converter for Unity 导入项目时可能遇到的常见问题。

## 问题列表

### 问题 1：Figma 请求限制

**症状**：返回 429 Too Many Requests 错误

**原因**：Figma 对免费账户应用请求限制

**解决方案**：
- 等待冷却期（约 4 天）
- 升级到 Figma Full Seat 订阅
- 使用 ZIP 副本导入方式

### 问题 2：图像未正确导入

**症状**：某些图像未被正确导入

**可能原因**：
- 父组件不符合自动合并规则
- 某些效果不被 Unity 组件支持

**解决方案**：
- 检查是否符合 "Layout Rules" 中的规则
- 使用 "img" 标签强制合并
- 使用 "cont" 标签强制分割

### 问题 3：文本渲染不一致

**症状**：Unity 中的文本显示与 Figma 不一致

**解决方案**：
- 启用 "Auto-Size" 选项
- 调整 TextMeshPro 字体文件的边距设置
- 使用 TextMeshPro 而非 UI.Text
