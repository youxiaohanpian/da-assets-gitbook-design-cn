---
title: 导入问题
description: D.A. Assets 设计师文档
---

1. **导入后我的 Frame 看起来和 Figma 布局不一样。为什么？**  
   答案可以在 **[布局规则](/layout-rules/)** 章节（《设计师手册》部分）中找到。

2. **我的组件被合并为单个图像，而我想把它们分开。**  
   **我的组件由多个图像组成，而我想将它们合并为一个。**  
   你会在《设计师手册》的 **"命名与标签"** 章节中找到此问题的解决方案。

3. **"Either this file doesn't exist or you don't have permission to view it. Ask the file owner to verify the link and/or update permissions"**  

   如果你看到此错误，需要阅读《设计师手册》中的 **[团队协作](/teamwork/)** 章节。  
   《设计师手册》的 **"团队协作"** 章节将**在所有图像都缺失时帮助你**。

4. **如果你看到这些错误，你可能已达到 API 请求限制。**

   ![API 限制错误](../files/xLKo0N9G2GIYxRtqxo0B.png)

   **你可能已达到 Figma API 请求限制**，这将暂时阻止你导入 Frame。  
   **为避免此问题**，请遵循以下准则：
   - 不要一次导入**超过 100 个** Frame；

   **如果你已达到限制，需要等待**一段时间才能再次导入 Frame，或者创建一个新项目。

   > **提示：** 这些不是要求，而是基于个人经验的建议。

5. **`ArgumentOutOfRangeException DA_Assets.FCU.CurrentProject.TryGetByIndex (at CurrentProject.cs)`**

   要解决此问题，你需要安装正确版本的 Json.NET。  
   通过 Package Manager 安装 Json.NET 的信息可以在本手册开头找到。
