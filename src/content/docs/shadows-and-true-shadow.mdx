> **提示：** 视频手册：https://www.youtube.com/watch?v=ckyS96RmsY8

从 2.0.0 版本开始，资产支持导入 Figma 阴影，因此在导入布局之前**不需要禁用**阴影组件。

如果 Figma 的阴影超出了包含阴影组件的容器，组件将会失真。  
为防止这种情况，请为你的 Frame **禁用** **"Clip content"（裁剪内容）** 功能。

![禁用 Clip Content](/files/WNJZsKQSk2EPZtDUkw5c.png)

如果你想使用 **"TrueShadow"** 资源，则**需要在导入布局之前禁用**组件的阴影，否则 **"TrueShadow"** 资源的阴影会复制 Figma 的阴影，并将其叠加。

![TrueShadow 阴影叠加](/files/3oxYzgBmulRxHYP4mBVa.png)
