1. **错误示例** — 如果你将来在 Unity 中通过算法或手动将此文本组件的值设置为 `"1000000000"`，它将无法容纳并超出边界。

   ![错误示例](/files/kDziw6z3Rgq0t7aWsdsC.png)

2. **正确示例** — 文本组件的正确尺寸设置。

   ![正确示例](/files/wDlmGzSZaEJTDZzIihng.png)

3. Figma 和 Unity 中的字体渲染原理不同，因此如果你使用 **TextMeshPro**，可能需要在 Unity 中对**字体文件**进行调整（例如边缘间距），以获得相同的外观。

   > **提示：** 如果你使用 **UI.Text**，则无法进行调整，因此建议使用 **TextMeshPro**。

4. 如果你不想对 Figma 布局或字体文件做任何更改，可以在资产设置的 **[文本与字体](/text/)** 选项卡中启用 **"Auto-Size"（自动调整大小）** 选项。此选项默认启用。

   在这种情况下，如果文本无法容纳，其大小将自动缩小。最大大小将是你在 Figma 中设置的大小。
