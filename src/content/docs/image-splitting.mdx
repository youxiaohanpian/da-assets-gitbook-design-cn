# Image Splitting（图像分割）

## 功能说明

如果您希望将父对象的子对象作为单独的对象导入，请对父元素名称应用 "**cont**" 标签。

## 对比示例

### 使用 cont 标签的效果：
- 子对象作为单独对象导入

### 不使用 cont 标签的效果：
- 子对象被合并为单个图像

## 组件名称示例

**带有标签的组件名称示例：**
```
cont - char - mario - lg-run
```

## 特殊规则

如果对象的某个子组件包含单词 "**icon**"，则**等同于**父组件具有 "**cont**" 标签。
