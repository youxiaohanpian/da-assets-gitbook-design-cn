# Image Merging（图像合并）

## 功能说明

如果您想要将父对象和子对象合并到单个组件中，请对父元素名称应用 "**img**" 标签。

## 对比示例

### 没有 img 标签的情况：
包含一张图片 (width="239")

### 有 img 标签的情况：
包含一张图片 (width="88")

## 示例

**带标签的组件名称示例：**
```
img - char - mario - lg-run
```

## 自动导入规则

如果父组件的所有子组件都是矢量图，父组件将**自动**作为单个图像导入。
