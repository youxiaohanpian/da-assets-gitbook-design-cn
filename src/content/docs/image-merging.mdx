# 图像合并

如果你希望将父对象和子对象合并为单个组件，请在父元素名称上应用 **`img`** 标签。

**不带 img 标签时：**

![不带img标签](/files/sktdbmBQX9G89k8mntGy)

**带 img 标签时：**

![带img标签](/files/uJioyg5SILxrTm6unqAa)

> **提示：** 带标签的组件名称示例：`img - char - mario - lg-run`

![组件名称示例](/files/pJbCT5kMlrggWoTik4u1)

> **提示：** 如果父组件的所有子组件都是矢量，父组件将自动作为单个图像导入。
