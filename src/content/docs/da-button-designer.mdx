# D.A. Button 设计师版

## 简介

D.A. Button 支持**多个** TargetGraphics，这意味着它们都可以在点击、悬停或禁用时**被动画化**。
设置状态的原理与 **UI.Button** 相同（参见 **UI.Button** 部分），但现在你可以为无限数量的 TargetGraphics 配置类似的行为。
