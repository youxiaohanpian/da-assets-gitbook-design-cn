如果你需要界面在 Unity 的不同分辨率下正确显示，需要配置 **Constraints（约束）**。

在 Unity 中，**Constraints** 称为 **Anchors（锚点）**，其工作方式与 Figma 完全相同。

![Figma 约束与 Unity 锚点](../files/gwDnnc07i3soeGjFU2zc.png)

唯一的例外是：在 Unity 中，你不能使用 **"Scale"（缩放）** 锚点；相反，你需要在 Figma 中使用 **"Left and Right"（左和右）** 或 **"Top and Bottom"（上和下）** 约束。

> **提示：** 你可以从本教程学习如何在 Figma 中设置约束：  
> https://www.youtube.com/watch?v=U8N1gLcnATE
