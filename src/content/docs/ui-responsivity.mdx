# UI Responsivity（UI 响应性）

## 概述

本页面介绍如何在 Unity 中实现 UI 的响应式布局。

## 响应式设计原则

### 1. 使用 Anchors（锚点）

在 Figma 中设置好布局后，Unity 会自动将 Anchors 应用到导入的对象上。

### 2. 使用 Constraints（约束）

Figma 中的 Constraints 会自动转换为 Unity 中的 RectTransform 约束设置。

**Constraints 映射：**

| Figma Constraint | Unity RectTransform |
|----------------|-------------------|
| Left | Left |
| Right | Right |
| Top | Top |
| Bottom | Bottom |
| Center | Middle / Center |
| Scale | Scale with screen size |

### 3. Safe Area（安全区域）

对于移动设备，考虑使用 Safe Area 来避免屏幕缺口和圆角。

## 最佳实践

- 在 Figma 中正确设置 Constraints
- 使用 Scale with screen size 作为 Canvas Scaler 模式
- 测试不同分辨率和宽高比
