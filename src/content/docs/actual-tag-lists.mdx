## 标签说明

| 标签 | 说明 | 适用框架 |
|------|------|---------|
| `img` | 将组件标记为图像 | Canvas, UITK, Nova |
| `cont` | 容器标签 | Canvas, UITK, Nova |
| `btn` | 将组件标记为按钮 | Canvas |
| `ignore` | 忽略此组件 | Canvas, UITK, Nova |
| `scroll` | 滚动视图相关 | Canvas |
| `fld` | 输入字段 | Canvas |
| `pwd` | 密码字段 | Canvas |
| `ph` | 占位符 | Canvas |
| `tg` | 触发器 | Canvas |
| `tgr` | 触发器根 | Canvas |
