# Ignore (tag)（忽略标签）

## 功能说明

```
ignore - my object
```

**作用**：忽略当前对象及其所有子对象（该对象将不会被导入）。

**等同功能**：该标签等同于在 Figma 层级中禁用对象可见性（关闭眼睛图标）。

## 使用场景

- 暂时不想导入某个对象或组件
- 在 Figma 中隐藏某些元素而不删除
- 临时跳过某些组件的导入

## 示例

```
ignore - temp element
ignore - draft-button
ignore - work-in-progress
```

> 📎 **提示**：使用 ignore 标签与使用 Figma 的隐藏功能效果相同。
